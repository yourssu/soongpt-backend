## 📖 용어 설명

| 용어       | 의미           | 설명                      |
|----------|--------------|-------------------------|
| **부필**   | 부전공 필수       | 부전공생이 반드시 이수해야 하는 과목    |
| **부선**   | 부전공 선택       | 부전공생이 선택적으로 이수할 수 있는 과목 |
| **부전공**  | Minor        | 21학점 이수, 학위는 주전공만 취득    |
| **복수전공** | Double Major | 36~45학점 이수, 학위 2개 취득    |

## 📝 참고사항

1. **부전공 vs 복수전공 차이**
    - 부전공: 21학점 이수, 학위 미취득, 졸업장에 표기
    - 복수전공: 36~45학점 이수, 학위 2개 취득
    - 부전공이 상대적으로 진입장벽이 낮고 이수 부담이 적음

2. **부전공 신청 자격**
    - 1학기 이상 이수한 학생
    - 평점 및 취득학점 제한 없음 (단, 학과별로 제한 가능)
    - 복수전공보다 경쟁률이 낮음

3. **이수 요건**
    - 일반적으로 21학점 이수
    - 부필 과목이 있는 경우 반드시 이수
    - 부선 과목으로 나머지 학점 채움

4. **신청 방법**
    - u-SAINT 포털 → 학사관리 → 학적변동 → 다전공 신청
    - 매 학기 신청 가능 (연 4회)
