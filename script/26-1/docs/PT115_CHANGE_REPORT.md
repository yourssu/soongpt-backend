# PT-115 변경사항 최종 보고서

## 1) 목적

26-1 데이터 기준으로 course/target 매핑 품질을 점검하고,
실제 DB target 정합성 보정 및 리뷰 결과 정리를 수행함.

## 2) 주요 수행 내역

### A. DB target 직접 반영/보정

- 벤처경영학과(계약학과) `raw target=전체` 누락 10건 반영 후 scope 보정
  - 최종: `DEPARTMENT(벤처경영학과)`
- SW교육팀 Co-op 2건 보정
  - `2150059401`, `2150156801`
  - 단순 UNIV 반영분 제거 후 분류 기반 DEPARTMENT 다중 매핑 반영
- 경영학과 개설 + `전체` 누락 12건 반영
  - DB 기준 `경영학부` 매핑
- 국제무역학과 관련 4건 반영
- 정보통계·보험수리학과 3건 반영
- 전기공학과 5건 반영
  - DB 기준 `전기공학부` 매핑
- 차세대반도체학과 12건 보정
  - grade 기반 `UNIVERSITY` row 유지
  - 중복/불일치 유발하던 `DEPARTMENT` row 25건 삭제
- 유형 B 잔여 4건 보정
  - `2150155001`, `2150386604`, `2150465601`, `2150663306`
- 기초공학수학1 2개 분반(`5006762501`, `5006762502`) 보정
  - 잘려있던 `course.target` 원문 복구
  - target 정합성 검증 결과: `missing=0`, `extra=0`

### B. 파서 룰 보강
