## 📚 교직과정 개요

교직과정은 교원자격증 취득을 위한 필수 과정으로, **교직이론**, **교직소양**, **교육실습** 3개 영역으로 구성됩니다.

### 🎯 총 이수학점
- **교직이론**: 12학점 이상
- **교직소양**: 6학점 이상
- **교육실습**: 4학점 이상
- **합계**: 최소 22학점

### 3️⃣ 교육실습 (4학점 이상)

실제 학교 현장에서 교육 실습을 수행하는 영역입니다.

#### ⚠️ **주의: CSV에 미포함**

```
❗ 학교현장실습, 교육봉사활동 과목은

→ 별도 신청 및 관리 (학사팀 교직부)
```

#### 필수 이수 항목

| 항목 | 학점 | 이수시기 | 관리부서 |
|------|------|---------|---------|
| **학교현장실습** | 2학점 | 4학년 1학기 | 학사팀 교직부 |
| **교육봉사활동** | 2학점 | 2-4학년 | 학사팀 교직부 |

**이수 방법:**
- **2항목 모두 필수** (총 4학점)
- 일반 수강신청이 아닌 **별도 신청 절차** 필요
- 학사팀 교직부에 직접 문의 및 신청

#### 📋 학교현장실습 상세

**실습 내용:**
- 수업 참관 및 수업 실습
- 학급 경영 참여
- 생활지도 실습
- 교무 업무 경험

**실습 절차:**
1. 실습학교 배정 (교육청 협조)
2. 사전 오리엔테이션 참석
3. 4주간 현장 실습 (4학년 1학기)
4. 실습일지 작성
5. 지도교사 평가
6. 실습 보고서 제출

**평가 방법:**
- Pass/Fail (P/F)
- 출석률 80% 이상 필수
- 지도교사 평가 70% 이상

#### 📋 교육봉사활동 상세

**활동 내용:**
- 초·중·고등학교 학습 지도
- 방과후 학교 보조
- 교육 관련 봉사활동
- 지역아동센터 등 교육기부

**인정 기준:**
- 총 60시간 이상
- 학기당 20~30시간 권장
- 공식 확인서 필수
- **학사팀 교직부에 사전 승인** 받아야 인정

**활동 인정 기관:**
- 초·중·고등학교
- 지역아동센터
- 교육청 인정 기관
- 대학 부설 교육기관

**⚠️ 중요:**
- 활동 전 반드시 학사팀 교직부에 문의
- 승인받지 않은 활동은 시간 인정 불가
- 확인서 양식은 학사팀에서 배부

## ⚠️ 주의사항

### 1. 이수 학점 관리
```
❗ 교직과정 학점은 졸업학점에 포함되지 않음
   → 졸업 요구학점 + 교직 22학점 별도 이수
```

### 2. 개설 과목 확인
```
❗ 학기별로 개설 과목이 다를 수 있음
   → 수강신청 전 u-SAINT에서 반드시 확인
   → 미개설 과목은 다음 학기에 이수 계획
```

### 3. 교육실습 별도 신청
```
❗ 학교현장실습, 교육봉사활동은 별도 신청
   → 학사팀 교직부에 직접 문의 및 신청
   → 일반 수강신청 시스템에 없음
```

### 4. 교직 적성 및 인성 검사
```
❗ 교직과정 이수 중 2회 실시
   → 1회: 2학년 (선발 시)
   → 2회: 4학년 (졸업 전)
   → 모두 적격 판정 받아야 자격증 취득 가능
```

### 5. 응급처치 및 심폐소생술 실습
```
❗ 교원자격증 취득 필수 조건
   → 2시간 이상 이수
   → 졸업 전까지 완료
   → 보건소 또는 대학 보건실에서 실시
```

### 6. 성적 기준
```
❗ 교직과목 평균 성적 80점 이상 (B° 이상)
   → 낮은 성적 과목은 재수강 필요
```

### 7. 전공 과목 이수
```
❗ 교과교육영역: 8학점 이상
❗ 교과내용영역: 50학점 이상 (2009년 이후 입학자 기준)
   → 전공 과목도 충분히 이수해야 함
```

## ⭐ 성공적인 교직과정 이수를 위한 팁

### 1. 계획적인 수강 신청
- **개설 과목 확인**: 매 학기 u-SAINT에서 교직과목 개설 여부 확인
- 미개설 과목은 다음 학기로 연기
- 전공 과목과 균형있게 배분
- 교육실습 전에 이론 과목 완료

### 2. 성적 관리
- 교직과목 평균 80점 이상 유지
- 낮은 성적은 과감히 재수강
- 전공 과목도 소홀히 하지 말 것

### 3. 교육봉사활동 조기 시작
- **사전 승인 필수**: 학사팀 교직부에 먼저 문의
- 2학년부터 시작하여 부담 분산
- 학기 중 매주 2-3시간씩 꾸준히
- 공식 확인서 반드시 받기
- 인정 기관 사전 확인

### 4. 교육실습 준비
- **별도 신청**: 일반 수강신청 아님, 학사팀 교직부에 문의
- 수업지도안 작성 연습
- 모의수업 경험 쌓기
- 교과내용 복습

### 5. 학사팀 교직부와 소통
- 정기적으로 이수 현황 확인
- 궁금한 사항은 즉시 문의
- 교육실습 관련 사항은 반드시 문의

### 6. 교원임용시험 대비
- 3학년부터 조금씩 준비
- 교직이론 복습

- 스터디 그룹 활용
