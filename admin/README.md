# SoonGPT Admin Panel

관리자용 과목 관리 웹페이지

## 개발 환경 실행

```bash
npm run dev
```

개발 서버가 http://localhost:3000 에서 실행됩니다.

## 빌드

```bash
npm run build
```

빌드된 파일은 `dist/` 디렉토리에 생성됩니다.

## 환경 변수

- `VITE_API_BASE_URL`: API 서버 URL (기본값: https://admin.soongpt.com)

## 기능

- 전체 과목 조회
- 페이지네이션
- 과목명/교수명 검색
