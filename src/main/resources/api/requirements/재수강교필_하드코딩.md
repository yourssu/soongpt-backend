2022학년도 이전 입학자 중 교과목 교양과목 교과목의 재수강을 원할 경우, 아래 교과목으로 재수강해야 함

* 독서와토론 → 인문적상상력과소통 교과목 내 ‘고전읽기와 상상력’
* 대학글쓰기 → 비판적사고와 표현 교과목 내 ‘미디어사회와 비평적글쓰기’
* 기업가정신과행동 → 창의적사고와 혁신 교과목 내 ‘혁신과 기업가정신’
* 현대인과성서 → 인간과 성서 교과목 내 ‘현대사회이슈와 기독교’
* 컴퓨터사고 → 컴퓨터적 사고 교과목 내 ‘컴퓨터적사고와 코딩기초’
* AI와데이터사회 → SW와 AI 교과목 내 ‘AI와 데이터 기초’
* Academic and Professional English 1, 2 → 재수강반 없음 (학점 포기 가능)

지금 발생한 문제가 뭐냐면 사용자 재수강해야하는 과목코드로 조회했는데 과목명중에 "현대인과성서"가 있는데 이 과목이 폐강되고, 이 과목을 재수강하려면 "현대사회이슈와 기독교" 과목을 들으면 현대인과성서 재수강으로 인정되기때문에 재수강과목리스트에 현대인과성서가 있다면 현대사회이슈와 기독교 를 반환해야해.
