### 23학번 이후

목적
교양선택 과목을 노출하기 전에, 사용자가 어떤 교양 선택 분야를 수강 완료했고, 미완료했는지 알기 위함

Input
사용자의 학번, 이때까지 들은 과목 리스트

Logic

1. 이때까지 사용자가 들은 과목들을 백과사전에 떄려 넣어서 수강 완료한 교양 선택 분야를 알아낸다. 한 과목이라도 수강했다면 해당 분야는 완료 처리를 한다.(성적 F 제외)
   1. 인간, 언어
   2. 문화, 예술
   3. 사회, 정치, 경제
   4. 과학, 기술 & Bridge 교과(수리, 물리, 화학, 생물)
   5. 자기개발, 진로탐색

Output

1. 5개 분야 중에 사용자가 완료한 분야 정보
   1. 인간, 언어 : 완료
   2. 문화, 예술 : 미완료
   3. 사회, 정치, 경제 : 완료
2. 5개 분야 중 3개 분야 이상을 수강 완료했는지 안했는지를 TRUE, FALSE로 보내준다
3. 사용자의 교양선택 이수현황(ex. 9학점 중 3학점 들었어요~~~~~)

### 20~22학번

목적
교양선택 과목을 노출하기 전에, 사용자가 어떤 교양 선택 분야를 수강 완료했고, 미완료했는지 알기 위함

Input
사용자의 학번, 이때까지 들은 과목 리스트

Logic

1. 이때까지 사용자가 들은 과목들을 백과사전에 떄려 넣어서 수강 완료한 교양 선택 분야를 알아낸다. 한 과목이라도 수강했다면 해당 분야는 완료 처리를 한다.(성적 F 제외)
   1. 그룹1: 숭실품성교과
      1. 인성과 리더십
      2. 자기계발과 진로탐색
   2. 그룹2: 기초역량교과
      1. 한국어의사소통
      2. 국제어문
   3. 그룹3: 균형교양교과
      1. 문학, 예술
      2. 역사, 철학, 종교
      3. 정치, 경제, 경영
      4. 사회, 문화, 심리
      5. 자연과학, 공학, 기술
2. 그룹 별 완료 여부를 체크한다.
   1. 그룹1 & 그룹2
      1. 그룹 중에 한 분야 이상이 완료됐다면, 그룹을 완료 처리한다.
   2. 그룹3
      1. 그룹 중에 두 분야 이상이 완료됐다면, 그룹을 완료 처리한다.

Output

1. 그룹 및 분야 별 완료 여부
   1. 그룹이 완료됐다면 내부 분야 완료 여부는 신경쓰지 않는다.
   2. 그룹이 완료되지 않았다면 내부 분야의 완료 여부를 보내준다
2. 사용자의 교양선택 이수현황(ex. 9학점 중 3학점 들었어요~~~~~)
