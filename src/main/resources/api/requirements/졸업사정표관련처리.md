지금 졸업사정표가 rusaint-service에서부터 null로 오는경우가 분ㅇ명있을텐데, 신입생1-1 은 아직 안나와서그런거란말이지.
  이거내가 일부러 에러처리안하고 1-1이구나 라고파악한건데, **현재 명세**에서는 그럴 경우에도 `progress`를 **null이 아닌** `required/completed=-2`, `satisfied=false` 로 보냅니다. 졸업사정표 데이터 자체가 없으니까 전기전필이 있는지조차 조회할 수 없어, "제공 불가"로 통일. → progress_프론트_가이드.md, 졸업사정표_경고_가이드.md
