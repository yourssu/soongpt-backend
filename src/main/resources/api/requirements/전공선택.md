목적
사용자가 이번 학기에 들을 수 있는 전공 선택 과목을 알기 위함

Input
토큰(사용자의 학번, 학과, 학년, 이때가지 수강한 과목, 졸업사정표의 전공선택 학점&총족 여부)

Logic

1. 이번 학기에 열린 사용자의 학과 전공선택 과목  + 타전공인정 과목 조회
2. 이미 수강한 과목 제외(성적 무관)
3. 남은 과목 학년별로 분리
   1. 1학년
   2. 2학년
   3. 3학년
   4. 4, 5학년
   5. 타전공인정

Output

1. 사용자의 전공 선택 과목 이수 현황(ex. 15/20)
2. 학년별로 분리된 전공선택 과목 목록
   1. 수강 과목 정보(과목 이름, 교수, 학점 `point`)-분반 전체 묶어서
   2. 수강해야하는 학년
   3. 대상외수강제한인 경우에는 태그 붙여서(수강 대상과 사용자가 일치하는 경우에는 제외)
3. 해당 학과의 타전공인정 과목
   1. 수강 과목 정보(과목 이름, 교수, 시간)-분반 전체 묶어서
   2. 수강해야하는 학년
   3. 대상외수강제한인 경우에는 태그 붙여서
   4. 개설 학과
4. 사용자의 현재 학년(초기 화면에서 몇학년을 선택한 상태로 보여줄지 고민)

Edge Case

* "졸업사정표 상 전공선택 충족 여부 == TRUE"일 경우 -> 이미 전공 선택 학점을 전부 채웠다는 메세지
  * TRUE 여부를 가장 먼저 검사하고 패스
