목적
사용자가 이번 학기에 들어야하는 교양 필수 과목과, 분야별 수강 내역을 알기 위해

### 23학번 이상

Input
토큰(사용자의 학번, 학과, 학년, 이때가지 수강한 과목, 졸업사정표의 전공필수 학점 충족 여부)

Logic

1. 이때까지 수강한 교양필수의 분야 조사….(여기도 교양선택처럼 매칭해야겠네요)
   1. 교양필수 분야 목록(분야 내부 상세과목명은 위 파일 확인 부탁드려요)
      1. 인문적상상력과소통(1학년)
      2. 비판적사고와표현(1학년)
      3. 인간과성서(1학년)
      4. 한반도평화와통일(1학년)
      5. 컴퓨팅적사고(1학년)
      6. SW와AI(1학년)
      7. 글로벌시민의식(2학년)
      8. 글로벌소통과언어(2학년)
      9. 창의적사고와혁신(3학년)
   2. 제가 23학년도 부터 쭉 찾아봤는데 23 24 25 모두 위 파일에 있는 과목명에서 변한 적은 없어서 그냥 텍스트로 매치해도 괜찮고, 과목명 앞에 분야가 붙어있어서 분야로 매칭해도 될 것 같습니다.
      1. 과목명 텍스트로 매칭할 경우에는 파일에 있는 과목명과 강의시간표의 과목명이 유사한지 유의부탁드립니다(ex. CTE의 경우에는 파일에만 Career Track English로 풀어져있음)
2. 놓친 분야가 있는지 확인(학기는 고려하지 않고 학년만 고려)
   1. 1학년
      1. x
   2. 2학년
      1. 인문적상상력과소통
      2. 비판적사고와표현
      3. 인간과성서
      4. 한반도평화와통일
      5. 컴퓨팅적사고
      6. SW와AI
      7. 위 분야 중 미수강 분야가 있다면 해당 분야를 전달(텍스트로)
   3. 3학년
      1. +2학년 목록에 있는 분야
      2. 글로벌시민의식
      3. 글로벌소통과언어
      4. 위 분야 중 미수강 분야가 있다면 해당 분야를 전달(텍스트로)
   4. 4학년
      1. +2학년 +3학년
      2. 창의적사고와혁신
      3. 위 분야 중 미수강 분야가 있다면 해당 분야를 전달(텍스트로)
   5. 걍 텍스트로 전달해주고, 프론트에서 잘 떼우겠습니다
3. 정상 분야 과목 파트
   1. 이번 학기에 수강 대상이 맞는 과목이 있는 "분야"및 과목 조회
   2. 이미 수강한 분야의 과목은 제외
   3. 그 과목들을 교양필수 분야로 묶어서 전달

Output

* 놓친 분야의 정보(텍스트로)
  * ex) 글로벌시민의식
* 이번 학기에 수강 대상이 맞는 교양 필수 과목
  * 과목 정보(과목 이름, 교수, 시간)-분반&분야로 묶어서
* 현재 교양필수 학점 이수 현황

Edge Case

* "졸업사정표 상 교양필수 총족 여부 == TRUE"일 경우 -> 이미 교양필수 학점을 전부 채웠다는 메세지
  * TRUE 여부를 가장 먼저 검사하고 패스
* "졸업사정표 상 교양필수 총족 여부 == FALSE" && 보내줄 과목 정보가 하나도 없는 경우(수강 대상 맞는 과목이 없는 경우)
  * 수강 대상이 맞는 과목이 없음을 알리는 메세지
  * sw와 ai 전체학년 1개 열리

### 22학번 이하

Input
토큰(사용자의 학번, 학과, 학년, 이때가지 수강한 과목, 졸업사정표의 전공필수 학점 충족 여부)

Logic

1. 이때까지 수강한 교양필수의 분야 조사(23학년도 이후에 생성된 과목만 해당)
2. 이번 학기 교양 필수 과목 조회
   1. 이번 학기에 수강 대상이 맞는 과목이 있는 "분야"및 과목 조회
   2. 이미 수강한 분야의 과목은 제외
   3. 이미 수강한 과목 제외
   4. 그 과목들을 교양필수 분야로 묶어서 전달

Output

* 이번 학기에 수강 대상이 맞는 교양 필수 과목
  * 과목 정보(과목 이름, 교수, 시간)-분반&분야로 묶어서
* 현재 교양필수 학점 이수 현황

Edge Case

* "rusaint에서 받아온 graduation summary satisfied== TRUE"일 경우 -> 이미 교양필수 학점을 전부 채웠다는 메세지
  * satisfied=true
* "rusaint에서 받아온 graduation summary satisfied== FALSE" && 보내줄 과목 정보가 하나도 없는 경우(수강 대상 맞는 과목이 없는 경우)
  * satisfied=false required > 0 인 경우
